# About
Implementation of data structures from [Open Data Structures](http://opendatastructures.org/) as a NodeJS module.

### Includes
- Array-Based Lists
    + Array Stack
    + Fast Array Stack
    + Array Queue
    + Array Deque
    + Dual Array Deque
    + Rootish Array Stack
- Linked Lists
    + Singly-Linked List
    + Doubly-Linked List
    + Space-Efficient Linked List
- Skiplists
    + SSet Skiplist
    + List Skiplist
- Hash Tables
    + Chained Hash Table

## Usage
```javascript
const ods = require('./ods.js')
```

## Test
```
npm test
```
